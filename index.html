Programming Languages, Terms and Concepts

This is the wiki page that lists, defines, and describes numerous programming terms and concepts we have learned at Epicodus. To learn more about a term, select one of the boxes below to view a description, sample code and a demonstration of how the code results appear.

A common language programmers learn at the beginning of their education is a language called Markdown. This is generally used to quickly create and edit files on your local drive. Programmers frequently use this language to develop their README pages that provide background details on their project.
