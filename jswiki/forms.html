Here is an example of a form:

Hello ! You are years old!

Here is what the HTML code looks like for the form:

<form id="form"> <label for="name">Your Full Name</label> <input type="text" name="name" value="name"> <label for="age">Your Age in Years</label> <input type="text" name="age" value="age"> </form>

Here is what the Javascript code looks like for the form:

          $(function(){
            $('#form').submit(function(event) {
              // debugger;
              var username = $('input.name').val();
              var userage = $('input.age').val();

              $('#reveal').show();
              $('#name').text(username);
              $('#age').text (userage);

            event.preventDefault();
            });
          });
